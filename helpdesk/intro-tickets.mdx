title: "Introduction to Tickets"
description: Help Desk APIs help with creating and tracking tickets.
---

The Solid Help Desk is how clients communicate with the Solid customer success team. For more information, please see the [Client Support section in the Help Center](https://help.solidfi.com/hc/en-us/articles/4414021119131-Overview-of-Client-Support).

Using the Help Desk API, you can:  
- [Create a Ticket](https://www.solidfi.com/docs/create-a-ticket)  
- [Update a Ticket](https://www.solidfi.com/docs/update-a-ticket)  
- [Retrieve a Ticket](https://www.solidfi.com/docs/retrieve-a-ticket)  
- [List all Tickets](https://www.solidfi.com/docs/list-all-tickets)  
- [Upload an Attachment](https://www.solidfi.com/docs/upload-an-attachment)  
- [Update a Ticket with comments](https://www.solidfi.com/docs/update-a-ticket)  
- [Delete a Ticket](https://www.solidfi.com/docs/delete-a-ticket)  
- [Add Ticket Comments](https://www.solidfi.com/docs/add-ticket-comments)  
- [List Ticket Comments](https://www.solidfi.com/docs/list-ticket-comments)

For a description of fields, see this [article on the Ticket Object](https://www.solidfi.com/docs/the-ticket-object).

---
title: "Metadata"
description: The Solid Platform supports metadata to add information to the resource
---
The Solid Platform support includes metadata on the following resources:
[Persons](https://www.solidfi.com/docs/the-person-object), [Businesses](https://www.solidfi.com/docs/the-business-object), [Accounts](https://www.solidfi.com/docs/the-account-object), [Contacts](https://www.solidfi.com/docs/the-contact-object), [Cards](https://www.solidfi.com/docs/the-card-object), [Transactions](https://www.solidfi.com/docs/the-transaction-object), and [Transfers](https://www.solidfi.com/docs/the-send-transfer-object)

Metadata takes the form of free-form key-value pairs. You may send metadata when you create an object (POST) and when updating the object (PATCH).  If you would like to remove metadata that is already on an object, you can unset it by passing in the key-value pair with an empty string, like this:  

```json
{  "key": "" }
```

Current limitations:  
- Max of 20 keys for any single metadata object  
- Max of 4KB for any single metadata object

---
title: "Environments"
description: Solid provides both LIVE and TEST environments.
---
For ease of development, Solid provides both LIVE and TEST environments. The dashboard displays the "LIVE/TEST" toggle to switch between both environments. TEST and LIVE environments are identical except that no real KYC, KYB, and transactions are made in the TEST environment. TEST environment speeds up your FinTech product development.
 

```url TEST
https://test-api.solidfi.com
```


```url LIVE
https://api.solidfi.com
```
